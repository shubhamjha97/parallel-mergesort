# parallel-mergesort

### Compilation:
1. OpenMP
```bash
make omp
```
2. CUDA
```bash
make cuda
```
3. Sequential
```bash
make seq
```

### Running
1. OpenMP
```bash
./mergesort_omp <number-of-elements>
```
2. CUDA
```bash
./mergesort_cuda <number-of-elements>
```
4. Sequential
```bash
./mergesort_seq <number-of-elements>
```
